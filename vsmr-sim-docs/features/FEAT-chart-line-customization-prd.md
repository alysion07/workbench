# PRD: Simulation 모니터링 그래프 라인 커스터마이징

**문서 버전**: 1.0  
**작성일**: 2026-04-01  
**상태**: Draft  

---

## 1. 배경 및 문제 정의

### 1.1 현재 상황
VSMR 시뮬레이터의 Simulation 모니터링 대시보드(`DynamicChartGrid`)는 Minor Edit 설정 기반으로 차트를 자동 생성하며, 각 차트(`ChartCard`)는 Recharts `<LineChart>`로 다중 라인을 표시한다.

현재 차트의 라인은 다음과 같은 제약이 있다:
- **색상**: `generateColor()` 함수로 인덱스 기반 자동 할당, 사용자 변경 불가
- **선 스타일**: 모든 라인이 `solid` 고정 (`strokeWidth: 2`, `strokeDasharray` 없음)
- **가시성**: 모든 라인이 항상 표시, 개별 숨김/표시 불가
- **강조**: 특정 라인을 강조하여 비교 분석하는 기능 없음

### 1.2 문제
시뮬레이션 결과를 분석할 때, 하나의 차트에 여러 라인이 겹쳐 표시되면:
- 관심 있는 변수만 골라보기 어렵다
- 비슷한 색상의 라인이 겹치면 구분이 안 된다
- 특정 라인을 강조하여 추세를 파악하기 어렵다
- 발표/보고용으로 차트를 캡처할 때 시각적 커스터마이징이 불가하다

### 1.3 목표
사용자가 차트의 각 라인을 **개별적으로 제어**할 수 있도록 하여, 시뮬레이션 결과 분석의 효율성과 가독성을 높인다.

## 2. 사용자 시나리오

### 시나리오 1: 특정 변수에 집중 분석
> 엔지니어가 5개 라인이 있는 Pressure 차트에서 특정 Volume 2개의 압력 변화만 비교하고 싶다.  
> → Legend에서 나머지 3개 라인을 더블클릭하여 숨기고, 관심 있는 2개 라인만 확인한다.

### 시나리오 2: 라인 강조로 추세 파악
> 차트에서 특정 라인의 추세가 이상해 보인다. 다른 라인과의 관계를 빠르게 파악하고 싶다.  
> → Legend에서 해당 라인을 클릭하면 굵게 강조되고, 나머지는 반투명 처리되어 비교가 쉬워진다.

### 시나리오 3: 시각적 구분 개선
> 자동 할당된 색상이 비슷하여 라인 구분이 어렵다.  
> → Legend 옆 설정 아이콘을 클릭하여 색상을 직접 변경하고, 일부 라인은 점선으로 바꿔 구분한다.

### 시나리오 4: 보고서용 차트 정리
> 시뮬레이션 결과를 캡처하여 보고서에 첨부해야 한다.  
> → 중요한 라인은 굵게, 참고 라인은 얇은 점선으로 설정하고, 불필요한 라인은 숨긴 뒤 캡처한다.

## 3. 기능 요구사항

### FR-1. 라인 보이기/숨기기 토글

| 항목 | 내용 |
|------|------|
| **UI** | Recharts `<Legend>` 컴포넌트 활용 |
| **트리거** | Legend 항목 **더블클릭** |
| **동작** | 해당 라인 숨김 ↔ 표시 토글 |
| **시각 피드백** | 숨겨진 항목: Legend 텍스트에 취소선(`line-through`) + 반투명(`opacity: 0.4`) |
| **기본 상태** | 모든 라인 표시 |
| **제약** | 마지막 1개 라인은 숨길 수 없음 (최소 1개 표시 보장) |

### FR-2. 라인 강조 (Highlight)

| 항목 | 내용 |
|------|------|
| **트리거** | Legend 항목 **클릭** (토글 방식) |
| **강조 효과** | 선택된 라인: `strokeWidth` 현재값의 2배 |
| **비강조 효과** | 나머지 라인: `strokeOpacity: 0.2` |
| **해제** | 같은 항목 다시 클릭 시 강조 해제 → 모든 라인 원복 |
| **복수 선택** | 여러 라인 동시 강조 가능 |
| **강조 없을 때** | 모든 라인 기본 스타일로 표시 |

### FR-3. 색상 변경

| 항목 | 내용 |
|------|------|
| **진입점** | Legend 항목 옆 설정 아이콘 (팔레트 아이콘) 클릭 → 팝오버 |
| **UI** | 색상 피커 (자유 선택, hex/rgb 입력 지원) |
| **적용** | 즉시 반영: 차트 라인 색상 + Legend 아이콘 색상 동기화 |
| **기본값** | 기존 `generateColor()` 할당 색상 |

### FR-4. 선 스타일 변경

| 항목 | 내용 |
|------|------|
| **진입점** | FR-3과 동일한 팝오버 내 |
| **옵션** | 3종 프리셋 |
| **기본값** | 실선 (`solid`) |

| 스타일명 | 시각 표현 | `strokeDasharray` 값 |
|----------|-----------|----------------------|
| Solid (실선) | ─────── | `undefined` (없음) |
| Dotted (점선) | ·········· | `"2 2"` |
| Dashed (대시) | ── ── ── | `"6 3"` |

### FR-5. 선 굵기 변경

| 항목 | 내용 |
|------|------|
| **진입점** | FR-3과 동일한 팝오버 내 |
| **옵션** | 3단계 프리셋 |
| **기본값** | 보통 (2px) |

| 단계 | 라벨 | `strokeWidth` 값 |
|------|------|------------------|
| 얇은 | Thin | `1` |
| 보통 | Normal | `2` |
| 굵은 | Bold | `3` |

## 4. 비기능 요구사항

### NFR-1. 성능
- 라인 스타일 변경 시 차트 전체 리렌더가 아닌, 해당 `<Line>` 컴포넌트만 업데이트
- 팝오버 열림/닫힘 시 차트 데이터 재계산 없음

### NFR-2. 호환성
- 기존 기능(Y축 토글, 차트 크기 변경, 즐겨찾기, 비교 모드, 드래그/리사이즈)에 영향 없음
- `ResizableChartCard`의 기존 컨트롤 그룹과 자연스럽게 공존

### NFR-3. 반응형
- 차트 크기(small/medium/large)에 관계없이 Legend 및 설정 UI 정상 동작
- 작은 차트에서는 Legend가 스크롤 가능하도록 처리

## 5. UI/UX 상세

### 5.1 Legend 인터랙션 매핑

```
Legend 항목:
┌──────────────────────────────────────┐
│  ── Label Text          [⚙]         │
│  (색상 표시)  (라벨)   (설정 아이콘) │
└──────────────────────────────────────┘

- 클릭(Click)       → FR-2: 강조 토글
- 더블클릭(DblClick) → FR-1: 숨김/표시 토글
- [⚙] 클릭          → FR-3/4/5: 스타일 팝오버 열기
```

### 5.2 스타일 팝오버 레이아웃

```
┌─────────────────────────────┐
│  Line Style Settings        │
│─────────────────────────────│
│  Color                      │
│  ┌─────────────────────┐    │
│  │   [Color Picker]    │    │
│  └─────────────────────┘    │
│                             │
│  Style                      │
│  ( Solid ) (Dotted) (Dashed)│
│                             │
│  Width                      │
│  ( Thin ) (Normal) ( Bold ) │
└─────────────────────────────┘
```

### 5.3 Legend 상태별 시각 표현

| 상태 | Legend 텍스트 | 라인 표시 |
|------|-------------|----------|
| 기본 (표시중) | 일반 텍스트 | 기본 스타일 |
| 강조됨 | **볼드** 텍스트 | 굵게 + 나머지 반투명 |
| 숨겨짐 | ~~취소선~~ + 반투명 | 차트에서 제거 |

## 8. 수용 기준 (Acceptance Criteria)

- [ ] Legend 항목 **클릭** 시 해당 라인이 강조되고, 나머지 라인이 반투명(`opacity: 0.2`) 처리된다
- [ ] Legend 항목 **더블클릭** 시 해당 라인이 숨겨지고, Legend에 취소선이 표시된다
- [ ] 마지막 1개 라인은 숨길 수 없다
- [ ] Legend 옆 설정 아이콘 클릭 시 팝오버가 열린다
- [ ] 팝오버에서 색상 피커로 라인 색상을 자유롭게 변경할 수 있다
- [ ] 팝오버에서 선 스타일(실선/점선/대시)을 변경할 수 있다
- [ ] 팝오버에서 선 굵기(얇은/보통/굵은)를 변경할 수 있다
- [ ] 변경한 스타일이 차트 라인과 Legend에 즉시 반영된다
- [ ] 기존 차트 기능(Y축 토글, 크기 변경, 즐겨찾기, 비교 모드)이 정상 동작한다
- [ ] 여러 라인 동시 강조가 가능하다
