## 2. 무해한 차이 (MARS 동작 동일)

### 2.1 ICV 인라인 주석 제거 (624건)
GUI Export는 인라인 주석을 포함하지 않음. 데이터 값은 동일.
```
ICV:  200010000  0  101  1  0.930  1  * SMART330
PRI:  200010000  0  101  1  0.93   1
```

### 2.2 과학적 표기법 차이 (5건)
ICV는 Fortran 스타일(`1.69348-6`), PRI는 표준(`1.69348e-6`). 값 동일.
```
ICV: 1.69348-6     PRI: 1.69348e-6    (= 0.00000169348)
ICV: 7.93-5        PRI: 7.93e-5       (= 0.0000793)
ICV: 4.2337-9      PRI: 4.2337e-9     (= 0.0000000042337)
```

### 2.3 Junction 카드 병합 — CCC0101+0102 → 한 줄 (32건)
ICV는 CCC0101 (geometry)과 CCC0102 (loss coefficients)를 별도 줄에 작성.
PRI는 CCC0101 한 줄에 모든 필드를 병합 (CCC0102 카드 없음).
MARS는 "마지막 카드 우선" 규칙으로 동일하게 처리.
```
ICV:
  1150101  110010002  140010001  0.141145
  1150102  07.0  07.0  00000000

PRI:
  1150101  110010002  140010001  0.141145  7.0  7.0  00000000
```

### 2.4 ICV 중복 카드 (30건)
ICV에 동일 카드 번호가 2개 존재하는 경우 (마지막만 유효).
PRI는 파서가 deduplication 처리하여 유효 카드만 출력.
```
ICV:
  1150102  10.0  10.0  00000000   ← 무시됨
  1150102  07.0  07.0  00000000   ← 유효

PRI:
  (1150101에 7.0 7.0 병합됨)
```

### 2.5 MTPLJUN 초기조건 shorthand vs 개별 전개 (2건)
ICV는 shorthand로 여러 접합부에 동일값 적용, PRI는 개별 카드로 전개.
```
ICV:  6611011  0.0  0.0  5       ← 접합부 1~5 모두 적용
PRI:  6611011  0.0  0.0  1       ← 접합부 1
      6611021  0.0  0.0  2       ← 접합부 2
      6611031  0.0  0.0  3       ← 접합부 3
      6611041  0.0  0.0  4       ← 접합부 4
      6611051  0.0  0.0  5       ← 접합부 5
```

### 2.6 Card 115 (비응축 기체 질량 분율) — PRI에만 존재
ICV에는 없음. Card 110에 `air` 1종만 지정된 경우 기본값 = 1.0.
PRI가 기본값을 명시적으로 출력.
```
PRI:  115  1.0000
```

### 2.7 Card 200 (초기 시간) — PRI에서 제거됨
원본 PRI-SYS에는 `200  0.0`이 있었으나 ICV에 없으므로 수정본에서 제거.

### 2.8 Pipe shorthand vs 개별 카드
ICV는 `CCC0601  90.0  4` (볼륨 1~4에 90도 적용), PRI는 개별 카드로 전개.
PRI에만 있는 카드: `6200601`, `6200701`, `6340701`, `7200601`, `7200701` 등.

### 2.9 Control Variable 연속 카드 분리
ICV는 한 줄에 여러 쌍을 작성, PRI는 쌍마다 별도 줄.
```
ICV: 20599802  1.0  turpow  612  1.0  turpow  614
PRI: 20599802  1.0  turpow  612
     20599603  1.0  turpow  614
```
